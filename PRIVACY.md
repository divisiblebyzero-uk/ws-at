# Privacy Policy for Divisible By Zero Aural Trainer

**Last Updated: July 2026**

This privacy policy governs your use of the mobile application "Divisible By Zero Aural Trainer" (hereafter referred to as "the Application"), which was created by Divisible By Zero as an open-source educational tool.

## 1. Summary of Our Data Policy
We believe in absolute user privacy. The Application is an educational tool designed to function entirely offline. It does not collect, monitor, track, store, or transmit any user data, personal information, performance metrics, or telemetry.

The source code of the Application is publicly hosted under the AGPL-3.0 license at:
https://github.com/divisiblebyzero-uk/ws-at

For more information about our active development projects, please visit:
http://divisiblebyzero.co.uk/projects/

## 2. Information Collection and Use
* **Personal Data:** The Application does not collect any personally identifiable information (such as name, email address, phone number, or location data).
* **Device Permissions:** The Application requires no special hardware device permissions (such as access to your camera, microphone, storage disk, or contacts).
* **Audio Engine:** Audio playback is generated locally using your device's native Web Audio API via pre-rendered piano sample files and Tone.js script execution. No background ambient sounds are monitored, recorded, or sent.

## 3. Third-Party Data Access
Because the Application does not collect any user data, it is impossible for us to share or disclose any information to third-party companies, advertising networks, analytics platforms, or marketing entities.

## 4. Children’s Privacy
The Application is an educational utility designed for students of all ages preparing for advanced music theory examinations. We do not collect any personal data from anyone, including children under the age of 13.

## 5. Changes to This Privacy Policy
We may update this Privacy Policy from time to time. Any changes will be posted directly within this repository with an updated modification date.

## 6. Contact Us
If you have any questions or feedback regarding privacy while using the Application, please contact us via our project space at http://divisiblebyzero.co.uk/projects/ or open an issue directly in our source repository.
